The box model includes Margin, Border, Padding and then the padding. The Blue box to the right is 100 x 100 with no effects of the box model on. As we move down the list, you will see the different affects.

Content

Images and text go here. Things like padding affect the texts distance from the edge. As you can see, the box now has text in it.

Text

Padding

This adds padding between the border and the content to give a gap. The text now is spaced from the edge with a padding value of 10 pixels.

padding: 10px;

Text

Border

This surrounds both the content and the padding. In the example, it’s a 3 pixel solid white border.

border: 3px solid white;

Text

Margin

The Margin is a transparent area that clears a space around all the previous. There is a 30 pixel margin around the box.

margin: 30px;

Text
